# System Requirements

* Windows PowerShell 5.1 and .NET 4.6.1+
* PowerShell Core 6+

# Installing

To install globally:

```powershell
Install-Module -Name 'Carbon.ScheduledTasks'
Import-Module -Name 'Carbon.ScheduledTasks'
```

To install privately:

```powershell
Save-Module -Name 'Carbon.ScheduledTasks' -Path '.'
Import-Module -Name '.\Carbon.ScheduledTasks'
```
